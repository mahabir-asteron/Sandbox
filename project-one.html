PROJECT OVERVIEW

KEEPING THE PROJECT CLEAR.

Renovation projects can generate debris and unwanted materials as work moves from one stage to the next. Without a practical cleanup plan, materials can quickly begin taking over valuable project space.

This Edmonton renovation cleanup project focused on creating one organized location for suitable project materials and supporting a more manageable work area.

A practical bin rental solution helped support the ongoing cleanup needs of the project.
